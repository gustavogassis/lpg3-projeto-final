Projeto LPG3
Como rodar
* Instalar dependências rodando composer install
* Configurar credencias do banco no arquivo App\Infrastructur\Database.php
* Executar migrations da pasta /database
* Subir servidor rodando php -S localhost:8000

Acessar endpoints que estão no collection.json, que estão no formato Postman Collections

As rotas estão definidas no arquivo index.php
